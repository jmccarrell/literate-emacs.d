** Embark  (act on candidate)

   In the minibuffer, =C-.= pops a menu of actions on the current
   candidate. Outside the minibuffer, acts on the symbol / link /
   file at point.

   | key   | command         |
   |-------+-----------------|
   | C-.   | embark-act      |
   | C-h B | embark-bindings |

   Useful patterns:
   - =C-. E= during =consult-ripgrep= → export matches to a buffer
   - =C-.= on a file candidate → rename, copy, open other window
   - =C-.= on a symbol in code → xref, occur, rename-symbol

** Corfu  (in-buffer completion popup)

   Popup appears automatically as you type (=corfu-auto=). For an
   explicit trigger use =C-M-i= (=completion-at-point=) — it works
   regardless of indentation. =TAB= completes only when the line is
   already correctly indented; in Python the indent command cycles
   indent levels and usually consumes =TAB= first, so don't rely on
   it there. In Python/Eglot buffers the candidates come from the
   language server (ty).

   The keys below apply while the popup is open:

   | key         | command                                   |
   |-------------+-------------------------------------------|
   | TAB / RET   | insert the highlighted candidate          |
   | M-n / M-p   | next / previous candidate                 |
   | C-g         | dismiss the popup                         |
   | M-SPC       | insert separator (orderless multi-token)  |
   | M-PgDn/PgUp | scroll the doc popup (corfu-popupinfo)    |

** Recent files & known projects  (manage the lists)

   Two separate lists: =recentf= tracks recent /files/; projectile's
   =projectile-known-projects= is what =C-c p p= offers. Most pruning
   now happens automatically (recentf on mode start, projectile on
   load), but to manage by hand:

   | command                            | what it does                          |
   |------------------------------------+---------------------------------------|
   | M-x recentf-edit-list              | mark recent /files/ to remove         |
   | M-x recentf-cleanup                | drop non-existent / excluded files    |
   | M-x projectile-remove-known-project | drop one project from C-c p p        |
   | M-x projectile-cleanup-known-projects | drop projects whose dir is gone    |
   | M-x projectile-clear-known-projects | wipe the project list entirely       |
   | C-h v recentf-list / projectile-known-projects | inspect either list raw  |

** Worktree navigation  (Magit)

   Worktrees live under =<repo>.worktrees/<branch-slug>=.  Use Magit as
   the starting point: it asks Git for registered worktrees, so a stale
   directory is never offered as a checkout.

   | task                        | key / command                  | result                                  |
   |-----------------------------+--------------------------------+-----------------------------------------|
   | open the current repo       | C-x g                          | Magit status                            |
   | open worktree actions       | Z                              | create, visit, or remove a worktree     |
   | create branch + worktree    | C-x g, then Z c                | proposes <repo>.worktrees/<branch-slug> |
   | add an existing branch      | C-x g, then Z b                | checks it out in a new worktree         |
   | visit a registered worktree | C-x g, then Z g                | opens its Magit status                  |
   | quick worktree selector     | M-x jwm/magit-switch-worktree | choose a registered tree and open Magit |
   | move from status to files   | RET / C-x b / C-x C-j          | file, buffer, or Dired navigation       |

   The location prompt stays editable for exceptions.  Magit proposes a
   slash-safe branch name, so =RND-123/foo= becomes =RND-123-foo= in the
   default directory name.

** Worktree teardown  (kill buffers, then remove)

   Kill the worktree's buffers *before* removal.  The dedicated helper works
   from anywhere and protects modified buffers; after a worktree is gone,
   projectile can walk up to the next enclosing =.git= and scope too broadly.

   | step | key / command                          | note                                      |
   |------+----------------------------------------+-------------------------------------------|
   | 1    | M-x jwm/kill-buffers-under-dir          | choose the worktree root; preserves edits |
   | 2    | C-x g, then Z k                          | remove through Magit; shell also works    |
   | 3    | M-x projectile-remove-known-project      | or let next-session cleanup remove it     |

   Recovery, if you already removed the worktree and have buffers on
   now-missing files: =M-x jwm/kill-buffers-of-missing-files=.

** Project search and replace

   Use this for project-wide replacements with review before saving:

   1. =M-s r= search with =consult-ripgrep=.
   2. =C-. E= export matches to a grep buffer.
   3. =C-c C-p= enter =wgrep= edit mode.
   4. Replace text with =M-%= or normal editing.
   5. =C-c C-e= apply edits to file buffers.
   6. Review and save explicitly, e.g. =C-x s=.

** Rust (tree-sitter, Eglot, and formatting)

   One time on each machine, install the Rust language-server prerequisites in
   a shell, then install the grammar from Emacs:

   #+begin_src shell
     rustup component add rust-analyzer rust-src
   #+end_src

   Run =M-x jwm/ensure-treesit-grammars= and restart Emacs.  Opening a =.rs=
   file should then show =rust-ts-mode= in the mode line and Eglot should start
   =rust-analyzer= for its Cargo project.

   | key     | command             | result                                      |
   |---------+---------------------+---------------------------------------------|
   | C-c r f | eglot-format-buffer | format the current buffer through Rustfmt   |

   This is intentionally not format-on-save.  Keep formatting policy in the
   Cargo project's =rustfmt.toml= / =.rustfmt.toml= and use =cargo fmt --
   --check= for project-wide validation.

** wrap-region and insert-pair

    wrap-region select a region, then any of these:

    #+BEGIN_SRC emacs-lisp
      (wrap-region-add-wrappers
       '(("(" ")")
         ("[" "]")
         ("{" "}")
         ("<" ">")
         ("'" "'")
         ("\"" "\"")
         ("‘" "’"   "q")
         ("“" "”"   "Q")
         ("*" "*"   "b"   org-mode)                 ; bolden
         ("*" "*"   "*"   org-mode)                 ; bolden
         ("/" "/"   "i"   org-mode)                 ; italics
         ("/" "/"   "/"   org-mode)                 ; italics
         ("~" "~"   "c"   org-mode)                 ; code
         ("~" "~"   "~"   org-mode)                 ; code
         ("=" "="   "v"   org-mode)                 ; verbatim
         ("=" "="   "="   org-mode)                 ; verbatim
         ("_" "_"   "u" '(org-mode markdown-mode))  ; underline
         ("**" "**" "b"   markdown-mode)            ; bolden
         ("*" "*"   "i"   markdown-mode)            ; italics
         ("`" "`"   "c" '(markdown-mode ruby-mode)) ; code
         ("`" "'"   "c"   lisp-mode)                ; code
         ))
    #+END_SRC

  recall registers are named: [a-z][A-Z][0-9], denoted as R in the keybindings

  | C-x r s R               | copy region into R                                                  |
  | C-u C-x r s R           | copy region into R; then delete it from buffer                      |
  | C-x r i R               | insert text from region R                                           |
  | M-x append-to-register  | append region to text in register R; with prefix delete from buffer |
  | M-x prepend-to-register | prepend                                                             |
  |-------------------------+---------------------------------------------------------------------|
  | C-x r r R               | copy region into register                                           |
  | C-x r i R               | insert rectangle                                                    |
  |-------------------------+---------------------------------------------------------------------|
  | C-x r w R               | save the state of selected frames windows to R                      |
  | C-x r f R               | save the state of all frames                                        |
  | C-x r j R               | restore window or frame positions; same as position                 |
  |-------------------------+---------------------------------------------------------------------|
  | C-x r n R               | store 0 in register                                                 |
  | C-u NUMBER C-x r n R    | store NUMBER in register                                            |
  | C-x r + R               | increment by 1                                                      |
